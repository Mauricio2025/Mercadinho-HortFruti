# Mercadinho Hotifrut

Bem-vindo ao Mercadinho Hotifrut, um projeto web simples para simular um mercado de produtos hortifrúti. Utilizando HTML, CSS e JavaScript, este projeto oferece uma experiência interativa de compras online.

## Funcionalidades

- **Catálogo de Produtos:** Explore uma variedade de produtos frescos de frutas, legumes e verduras.
- **Carrinho de Compras:** Adicione produtos ao seu carrinho de compras e visualize o total da compra.
- **Atualizações em Tempo Real:** As mudanças no carrinho são refletidas instantaneamente na interface.
- **Responsivo:** Desfrute de uma experiência consistente em dispositivos desktop e móveis.
